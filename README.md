##Тестовое задание (ЯСН)
**Приложение предоставляет статистический анализ данных датасета стримингового сервиса Spotify.**
***
1. Скачайте или клонируйте репозиторий с проектом на Ваш компьютер;
2. На компьютере должна быть установлена система контейнеризации Docker и сборщик проектов Gradle (либо система управления реляционными базами данных PostgreSql и Java не ниже 11 версии);
3. В папке с проектом найдите файл ***src/main/resources/application-example.yaml***
   1. Переименуйте файл в application.yaml и откройте его;
   2. **Заполните поля spring.datasource.username и spring.datasource.password (и имя и пароль - postgres);**
4. Выполните gradle build;
5. Проверьте свободность порта 8080, если порт занят, укажите любой свободный порт в файле application.yaml, или освободите порт 8080 на Вашей машине;
6. В случае, если на Вашей машине стоит Docker, в корневой папке проекта из терминала или консоли выполните команду <docker-compose up -d>, это запустит приложение
7. Если docker не установлен
   1. Создайте базу данных Postgresql с именем yasn_db на порту 5432;
   2. В папке, где Вы сгенерировали jar-архив из терминала или консоли выполните команду java -jar yasn-0.0.1-SNAPSHOT.jar (либо стартуйте приложение из среды разработки);
8. Клиентская часть приложения будет доступна по адресу <http://localhost:8080>
***
##REST-API

    GET 
    http://localhost:8080/?colname=<имя колонки>&year=<год>

**colname** - обязательный параметр, тот по которому запрашивается статистика
> допустимые значения: duration_ms, year, popularity, danceability, energy, key

**year** - необязательный параметр, год, по которому запрашивается статистика

###Коды ответов:
>1. 200 - ок, успешная обработка запроса;
>2. 404 - в случае, если colname указан неверно и данные не найдены;
>3. 5xx - непредвиденная ошибка на стороне сервера.
